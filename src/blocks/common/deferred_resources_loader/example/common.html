Загрузчик отложенных ресурсов.

Функция принимает ссылку или массив ссылок на js-скрипты или css-файлы, и возвращает промис, который будет разрешен после загрузки всех скриптов и стилей.

Повторные вызовы apressImport с одинаковый url не приведут к повторному запросу ресурсов (будет возвращен разрешенный промис).

Пример использования: 

    
      apressImport("path/to/script.js|style.css").then(() => {
        // Код который должен выполниться после загрузки скрипта
      });
